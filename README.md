
# Password Generator

Gerador de Senhas é um aplicativo da web que gera senhas aleatórias dentro do seu navegador. As funcionalidades incluem:

- Senhas nunca saem do seu dispositivo - todos os dados ficam no seu navegador e nenhuma solicitação é feita na rede.
- Geração de senhas usando aleatoriedade criptograficamente segura.
- Diferentes opções para controlar como as senhas são geradas, incluindo número de dígitos, número de símbolos e quais símbolos usar.




Gerador de Senhas construído com HTML, CSS e Javascript.
